# Graph/RAG Rule Extraction Demos

Static presentation demos for the rule extraction pipeline.

## Pages

- `index.html` - landing page with links to both demos
- `rag_pipeline_transfer_animation.html` - Burnaby demo
- `calgary_rag_pipeline_transfer_animation.html` - Calgary demo

Upload or deploy this repository as a static site. The demos depend on the local
JS data files and crop image folders in this repo, plus the online
`3d-force-graph` CDN.
